h2. Structuur van een presentatie

In LaTeX kun je presentaties maken met de document class @beamer@. Een lege beamer presentatie ziet er zo uit.

{% highlight latex %}

\documentclass{beamer}
\usetheme{Copenhagen}

\author{Bert}
\title{A tale of two primes}

\begin{document}

\end{document}

{% endhighlight %}

Om slides te maken gebruik je de @frame@-omgeving met als argument de title van je slide. Schrijf bijvoorbeeld (in de @document@-omgeving)

{% highlight latex %}

\begin{frame}
\titlepage
\end{frame}

\begin{frame}{Artikel}
Hier kun je tekst schrijven.
\end{frame}

\begin{frame}{Mathematica}
En nog meer tekst.
\end{frame}

{% endhighlight %}

Dit geeft een titelslide en twee slides met een kopje en wat tekst.

Om de informatie in je presentatie te ordenen gebruik je de @block@, @enumerate@ en @itemize@-omgevingen. De @block@-omgeving is een speciale @beamer@-omgeving die je als volgt gebruikt.

{% highlight latex %}

\begin{frame}{Artikel}

\begin{block}{Definitie}
Dit is een definitie.
\end{block}

\begin{block}{Stelling van Euclides}
Dit is een stelling.
\end{block}

\end{frame}
{% endhighlight %}


Als je de elementen van je slide een voor een wilt laten verschijnen gebruik je het commando @\pause@. Dit commando kun je bijna overal in de code plaatsen. Bijvoorbeeld tussen twee @block@-omgevingen

{% highlight latex %}

\begin{frame}{Artikel}

\begin{block}{Definitie}
Dit is een definitie.
\end{block}

\pause

\begin{block}{Stelling van Euclides}
Dit is een stelling.
\end{block}

\end{frame}

{% endhighlight %}


of in een @enumerate@

{% highlight latex %}

\begin{frame}{Titel}

\begin{enumerate}
\item Element
\pause
\item Element
\pause
\item Element
\end{enumerate}

\end{frame}

{% endhighlight %}
