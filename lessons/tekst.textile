h2. Tekstopmaak

h4. Spatiëring

De spatiëring tussen woorden wordt automatisch door LaTeX bepaald.  Het is tegen de LaTeXfilosofie om handmatig witruimte in te voegen. 


h5. Non-breaking space

Een *non-breaking space* is een spatie die niet vergroot kan worden en waarop de regel niet wordt afgebroken. Je voegt een non-breaking space in met @~@. Je gebruikt dit bijvoorbeeld om initialen bij de achternaam te houden:

{% highlight latex %}
O.~P.~Losser
{% endhighlight %}

h4. Woordafbreking

LaTeX breekt woorden aan het einde van de regel af als dat nodig is. Voor de juiste woordafbreking moet je het package @babel@ laden met de juiste taaloptie

{% highlight latex %}
\usepackage[dutch]{babel}
{% endhighlight %}

Met @\-@ kun je afbreekpunten in een woord aangeven. Bijvoorbeeld 

{% highlight latex %}
tekst\-verwerkings\-programma's
{% endhighlight %}

De punten met @\-@ zijn dan de enige punten waar kan worden afgebroken. Dit is handig voor woorden met speciale tekens, omdat LaTeX die niet automatisch afbreekt. Hetzelfde geldt voor woorden die een streepje bevatten. Gebruik dit alleen als een lang woord problemen geeft.

h4. Aanhalingstekens

Aanhalingtekens zet je met een _backtick_ @`@ en een apostrof @'@.

<div class="example">
'quote', "quote", "Een 'quote' in een quote"
</div>

{% highlight latex %}
`quote', ``quote'', ``Een `quote' in een quote''
{% endhighlight %}

h4. Speciale tekens

Er zijn twee manieren om speciale tekens in te voeren: via _unicode_ en via commando's.

h5. Unicode

Door 

{% highlight latex %}
\usepackage[utf8]{inputenc}
{% endhighlight %}

te gebruiken in de preamble, kun je veel speciale tekens direct in je LaTeX-code zetten zoals je dat in bijvoorbeeld Word gewend bent.

h5. Kleinder dan en groter dan: @<@, @>@

Standaard geven @<@ en @>@ niet de uitvoer die je vewacht, maar ¡ en ¿. Dit is een probleem met de _font encoding_. Je krijgt de juiste tekens door @\textless@ en @\textgreater@ te gebruiken. Of gebruik @\usepackage[T1]{fontenc}@.

h5. Math mode

In de volgende les zullen we _math mode_ zien. Er zijn commando's voor speciale tekens die alleen in math mode werken.

h4. Streepjes

LaTex kent drie soorten streepjes in tekst: het *afbreekstreepje* ==-==, de *en dash* – en de *em dash* —. Deze zet je met @-@, @--@, en @---@. Gebruik de _en dash_ om een bereik aan te geven, bijvoorbeeld 'rijen 1–10'. Gebruik de _em dash_ als _gedachtestreepje_: 'He was running, Watson—running desperately.' In het Nederlands is het gebruikelijk om de _en dash_ omgeven door spaties te gebruiken als gedachtestreepje: 'Jacqueline vertrok – zonder nog een woord te zeggen.'

h4. Ellips

Een *ellips* is een rij van drie puntjes die aangeeft dat tekst is weggelaten. Om de juiste spatiëring te krijgen zet je een ellips met @\ldots@.

h3. Speciale paragrafen

h4. Tekst letterlijk zetten

De @verbatim@-omgeving zet gegeven tekst letterlijk, dat wil zeggen, LaTeX negeert eventuele commando's in de tekst. Gebruik het bijvoorbeeld wanneer je programmacode wilt zetten:

{% highlight text %}
\begin{verbatim}
def gcd(a,b):
    """
    Algoritme van Euclides.
    """
    while a:
            a, b = b%a, a
    return b
\end{verbatim}
{% endhighlight %}

h4. url's

Het @hyperref@-package heeft het commando @\url@ om url's mee te zetten. Je laadt @hyperref@ met

{% highlight latex %}
\usepackage[hidelinks]{hyperref}
{% endhighlight %}

Een url zet je bijvoorbeeld met

{% highlight latex %}
\url{http://www.ctan.org}
{% endhighlight %}

h4. Citaten

Er zijn drie standaardomgevingen voor quotes.

- @quote@ := voor korte citaten of meerdere korte citaten

- @quotation@ := voor langere citaten met meerdere alinea's

- @verse@ := voor citaten waar regelafbreking belangrijk is, zoals poëzie. De regel breek je af met @\\@.

<h3 class="panel-title">Onthouden</h3>
</div>
<div class="panel-body">

* Bijzondere symbolen (kopjes, speciale symbolen, accenten, aanhalingstekens, opsommingen) voer je in met commando’s.
* Dubbele spaties worden genegeerd — hoe rommelig je ook tikt, LaTeX lijnt alles uit.
* Harde returns in je bronbestand worden genegeerd: een harde return telt als een gewone spatie.
* Lege witregels worden niet gezet als regels wit. In LaTeX telt een lege regel wit als een einde alinea (alinea wordt afgesloten, wordt gezet, LaTeX speurt vooruit naar materiaal voor de volgende alinea).
* Alinea’s worden automatisch ingesprongen. Dat is goed. In de LaTeX-context willen we dat precies zo hebben.
