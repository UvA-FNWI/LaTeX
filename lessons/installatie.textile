Je installeert LaTeX door een *distributie* te installeren. Een distributie is een collectie van packages en programma's. Wij raden de volgende distributies aan.

h5. Windows en Linux

Installeer "TeX Live":http://www.tug.org/texlive/. Op Linux kun je ook je package manager gebruiken; in Ubuntu doe je bijvoorbeeld @sudo apt-get install texlive@. Deze distributie bevat TeXWorks.

h5. MacOS

Installeer "MacTeX":http://www.tug.org/mactex/; dit is de Macversie van TeX Live.
